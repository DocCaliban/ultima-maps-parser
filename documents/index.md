# Ultima Asset Tools

This project provides tools to decode and render maps, tilesets and other data files from the classic Ultima series. The primary focus is on making file formats and decoding logic clear and accessible for future maintainers, researchers, and fans.

Quick links
- Getting started: `documents/getting-started.md`
- Tools and generators: `documents/tools/generate-crc.md`
- CI and CRC workflow: `documents/ci.md`
- Developer docs and architecture: `documents/developer/architecture.md`

Assets and outputs
- Data files (original Ultima assets) belong in `assets/ultima-3/`.
- Rendered PNGs are written to the `out/` folder when you run the render tools.

For developer-level detail, see the `documents/developer` folder.
